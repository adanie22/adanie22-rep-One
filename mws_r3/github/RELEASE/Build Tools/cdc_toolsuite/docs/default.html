Default¶

default Module¶

Included in all modules that connect to Agility, loads rest api connection details from settings.json file into global variable dictionary CFG.

Settings_json¶

The settings file has follwing format:

{
    "DEFAULT": "AWS_DEV",
    "CDC_DEV": {
        "USER": "USER",
        "PASSWD": "PASS",
        "BASE_URL": "https://127.0.0.1:8443/agility/api/current",
        "MAXN": 2
    },..
}
